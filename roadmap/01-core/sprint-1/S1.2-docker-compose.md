## Description

Add `infra/docker-compose.yml` with Postgres 15+ and pgvector extension, OTel Collector, and Jaeger. One command (e.g. `docker-compose up -d`) brings up everything needed for local run. Production-ready criterion: single-command local run (goals.md §4.5).

---

## Requirements

- [ ] Postgres 15+ with pgvector extension available.
- [ ] OTel Collector and Jaeger services defined; agent/API will export to them later (S1.10).
- [ ] Single command starts all (e.g. `docker-compose -f infra/docker-compose.yml up -d` or `make run`).
- [ ] Document in root README how to run (can be in S1.13).

---

## Checklist

- [ ] Create `infra/docker-compose.yml`.
- [ ] Add service: postgres, image with pgvector (e.g. `pgvector/pgvector:pg15`).
- [ ] Add service: otel-collector (config in infra or default).
- [ ] Add service: jaeger (all-in-one or collector + query + UI).
- [ ] Wire OTel Collector to export to Jaeger if needed.
