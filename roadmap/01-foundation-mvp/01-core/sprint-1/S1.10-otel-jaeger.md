## Description

Instrument agent and FastAPI with OpenTelemetry (traces, logs). Export to OTel Collector; traces visible in Jaeger. Structured logging (JSON or key-value). Production-ready: structured logging (goals.md §4.5); NF2 (all agent actions in traces).

---

## Requirements

- [x] Agent: create spans for each node (Triage, Investigate, Decide, Report) and for MCP tool calls.
- [x] FastAPI: spans for requests (e.g. /ingest, /health, run trigger).
- [x] Export to OTel Collector (configured in docker-compose S1.2); Collector forwards to Jaeger.
- [x] Logs: structured (JSON or key-value), not free-form only.
- [x] Report includes link to trace (Jaeger trace URL) — coordinate with S1.7.

---

## Checklist
